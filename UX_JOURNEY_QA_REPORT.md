## ⚠️ REMAINING ISSUES (5 Critical)

### 1. index.css Theme Class Mismatch 🔴
**Issue:** `index.css` uses `.clean-dark`/`.clean-light` selectors but ThemeContext applies `.dark`/`.light`

**Impact:** `--background`, `--foreground`, `--card` shadcn/ui variables won't respond to theme changes

**Fix:** Update index.css selectors to use `.dark` and `.light` classes

### 2. gray-400 Contrast Issues 🟠
**Files:** 
- `channels-config.ts:313`
- `FormatPreview.tsx:252,285,299,309`
- `Recommendations.tsx:57`

**Fix:** Upgrade all `text-gray-400` to `text-gray-500`

### 3. Button Touch Targets 🟠
**File:** `ui/button.tsx`
- Icon button: 36px (needs 44px)
- lg button: 40px (needs 44px)

**Fix:** Update `min-h-11` for icon and lg sizes

### 4. Spinner → Skeleton Consistency 🟡
**20+ pages** use spinners instead of skeleton loaders

**Files:** LearningPaths.tsx, CertificationsGenZ.tsx, VoiceInterview.tsx, TestSession.tsx, etc.

**Fix:** Replace `animate-spin` with `SkeletonLoader` component

### 5. aria-pressed on Selection Buttons 🟡
**Files:**
- `Onboarding.tsx:191-246` (role/channel selection)
- `VoicePractice.tsx:667-677` (mode toggle)

**Fix:** Add `aria-pressed={isSelected}` to toggle buttons
