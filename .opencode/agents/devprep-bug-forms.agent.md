# Bug Hunter: Form Validation

Find and fix form validation bugs in the DevPrep codebase.

## Scope
- client/src/

## Bug Types
- Missing validation
- Incorrect validation logic
- Submit handler issues
- Error display bugs

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
