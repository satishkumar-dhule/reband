# Bug Hunter: Animations

Find and fix animation/transition bugs in the DevPrep codebase.

## Scope
- client/src/

## Bug Types
- Janky animations
- Missing transitions
- Performance issues with animations
- Interrupt handling

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
