# Bug Hunter: CSS & Styling

Find and fix CSS and styling bugs in the DevPrep codebase.

## Scope
- client/src/
- Any styles/ directories

## Bug Types
- Broken layouts
- Responsive issues
- Z-index conflicts
- Missing styles

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
