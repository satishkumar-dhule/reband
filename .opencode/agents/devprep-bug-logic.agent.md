# Bug Hunter: Logic

Find and fix logic bugs in the DevPrep codebase.

## Scope
- All source files

## Bug Types
- Incorrect conditions
- Off-by-one errors
- Wrong operators
- Edge cases not handled

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
