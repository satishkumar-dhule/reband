# Bug Hunter: SEO

Find and fix SEO bugs in the DevPrep codebase.

## Scope
- client/src/
- index.html

## Bug Types
- Missing meta tags
- Broken links
- Crawl issues
- Indexing problems

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
