# Bug Hunter: Accessibility

Find and fix accessibility issues in the DevPrep codebase.

## Scope
- client/src/components/
- client/src/pages/

## Bug Types
- Missing ARIA labels
- Keyboard navigation issues
- Color contrast problems
- Focus management issues

## Process
1. Read files thoroughly
2. Identify issues
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
