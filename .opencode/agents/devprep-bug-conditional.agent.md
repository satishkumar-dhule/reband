# Bug Hunter: Conditional Rendering

Find and fix conditional rendering bugs in the DevPrep codebase.

## Scope
- client/src/

## Bug Types
- Wrong conditions
- Flash of unstyled content
- Logic errors
- Ternary abuse

## Process
1. Read files thoroughly
2. Identify bugs
3. Fix using edit tool
4. Report in format:

```
## BUG-FOUND: [file:line]
- Issue: [description]
- Fix: [what you changed]
```
